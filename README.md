# Queenix Fitness Center

Ladies-only gym management and member app.

## Files
- `index.html` — Login page (3 portals: Owner, Operations, Member)
- `member.html` — Member app (classes, coffee, salon, tracker, profile)
- `admin.html` — Management dashboard (owner & ops portals)
- `vercel.json` — Vercel routing config
- `manifest.json` — PWA manifest (installable on phones)

## Setup
1. Get your Supabase anon key from: Supabase → Settings → API
2. In each HTML file, replace `PASTE_YOUR_ANON_KEY_HERE` with your real key
3. Deploy to Vercel
4. In Supabase → Auth → URL Configuration, add your Vercel URL
